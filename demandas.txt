🔗 1. Criar o repositório no GitHub

Vai no 👉 GitHub

Clica em “New repository”
Dá um nome pro projeto
Pode deixar sem README (importante pra evitar conflito)
Clica em Create repository
💻 2. Ir até sua pasta local

No terminal (PowerShell, CMD ou terminal do VS Code):

cd caminho/da/sua/pasta

Exemplo:

cd C:\Users\Leonardo\Documents\meu-projeto
⚙️ 3. Inicializar o Git na pasta
git init

Isso transforma a pasta em um repositório Git.

📦 4. Adicionar os arquivos
git add .

Depois:

git commit -m "primeiro commit"
🔌 5. Conectar com o GitHub

Copia a URL do repositório (algo como):

https://github.com/seu-usuario/seu-repo.git

E roda:

git remote add origin URL_DO_REPOSITORIO

Exemplo:

git remote add origin https://github.com/leonardo/meu-projeto.git
🚀 6. Enviar para o GitHub

Se for a primeira vez:

git branch -M main
git push -u origin main
✅ Pronto

Agora tua pasta local está conectada ao GitHub 🎉
A partir daqui, o fluxo padrão é:

git add .
git commit -m "mensagem"
git push
⚠️ Problemas comuns (pra te poupar dor de cabeça)
Erro de autenticação → hoje o GitHub usa token, não senha
Repositório já tem README → pode dar conflito no push inicial
Branch master vs main → resolvido com git branch -M main
